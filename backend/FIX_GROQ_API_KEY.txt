GROQ API KEY ERROR - QUICK FIX
================================

ERROR:
❌ 401 Client Error: Unauthorized for url: https://api.groq.com/openai/v1/chat/completions

CAUSE:
GROQ_API_KEY is either missing or invalid in your .env file

FIX:
====

1. Get a FREE Groq API Key:
   - Go to: https://console.groq.com
   - Sign up (free)
   - Create API key
   - Copy the key

2. Add to .env file:
   - Open: backend/.env
   - Add line: GROQ_API_KEY=your_actual_key_here
   - Save file

3. Restart backend:
   - Stop the Flask server (Ctrl+C)
   - Run: python app.py
   - Look for: "AI features enabled ✓"

4. Test:
   - Try parsing a question again
   - Should work now!

NOTES:
- Groq is FREE (30 requests/minute)
- Uses Llama 3.3 model
- Much faster than OpenAI
- No credit card required
